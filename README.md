# Number Guessing Game - C#

A console-based number guessing game built using C#.

## Overview

The program generates a random number between 1 and 100.
The user must guess the number until they get it correct.

## Features

- Random number generation
- Input validation
- Loop until correct guess
- Feedback (Too high / Too low)
- Guess counter

## Technologies Used

- C#
- .NET Console Application

## What I Learned

- Working with Random class
- While loops
- Conditional logic
- User input handling
